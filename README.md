# Roblox MCP via Mobile

Hubungkan MCP Roblox Studio ke developer via Cloudflare Tunnel.

## Panduan Client

| Langkah | Tindakan |
|---------|----------|
| 1 | Double-click `setup.bat` |
| 2 | Klik **Yes** saat diminta administrator |
| 3 | Copy URL `https://xxx.trycloudflare.com` |
| 4 | Kirim URL ke developer |

## Prasyarat

- Roblox Studio versi terbaru
- Koneksi internet

## Struktur Repo

```
├── setup.bat          # Double-click, jalan otomatis
└── README.md          # Panduan ini
```

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| mcp.bat tidak ditemukan | Update Roblox Studio |
| Gagal download cloudflared | Nonaktifkan antivirus sementara |
| Cloudflare error | Restart PC, jalankan ulang |
