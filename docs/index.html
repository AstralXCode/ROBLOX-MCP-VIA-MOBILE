Setup HP — Roblox MCP via Mobile

Konfigurasi Termux + OpenCode untuk mengontrol Roblox Studio

Prasyarat

  • Termux — install dari F-Droid (bukan Play Store)
  • Koneksi internet — WiFi / 4G / 5G
1

Install Termux

Download dari F-Droid, install, lalu buka aplikasinya.

2

Jalankan script setup

Copy paste perintah ini ke Termux, lalu Enter:

curl -fsSL https://raw.githubusercontent.com/AstralXCode/ROBLOX-MCP-VIA-MOBILE/main/phone/setup.sh | bash
3

Masukkan URL dari client

Buka file konfigurasi:

nano ~/.config/opencode/opencode.json

Ganti GANTI_DENGAN_URL_CLOUDFLARE dengan URL yang dikirim client.

https://acak-acak.trycloudflare.com/mcp

Simpan: Ctrl+X → Y → Enter

4

Jalankan OpenCode

opencode
Pastikan PC client sedang menjalankan setup.bat. Jika URL berubah, minta URL baru dari client.
